# Определения тригонометрических функций **Для острого угла в прямоугольном треугольнике:** - $\tan \alpha = \dfrac{\sin \alpha}{\cos \alpha}$ - $\cot \alpha = \dfrac{1}{\tan \alpha}$ **Пример:** В треугольнике с катетами $3$ и $4$, гипотенузой $5$: $\sin \alpha = \dfrac{3}{5} = 0.6$, $\cos \alpha = \dfrac{4}{5} = 0.8$, $\tan \alpha = \dfrac{3}{4} = 0.75$. **Расширение на любые углы** — с помощью [[Единичная окружность и радианная мера]]. **Связь с другими понятиями:** - [[Основные тригонометрические тождества]] следуют из этих определений. - [[Графики тригонометрических функций]] строятся на основе значений $\sin$, $\cos$.
