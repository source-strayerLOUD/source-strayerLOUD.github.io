# Графики тригонометрических функций

**Синусоида $y = \sin x$:**
- Область определения: $\mathbb{R}$
- Область значений: $[-1, 1]$
- Период: $2\pi$
- Нули: $x = \pi k,\ k\in\mathbb{Z}$

**Косинусоида $y = \cos x$:**
- Сдвинута относительно синуса: $\cos x = \sin\left(x + \frac{\pi}{2}\right)$

**Тангенсоида $y = \tan x$:**
- Период $\pi$, разрывы в $x = \frac{\pi}{2} + \pi k$, значения от $-\infty$ до $+\infty$

**Примеры:**
- График $y = 2\sin x$ — растяжение по вертикали (см. [[Преобразования графиков (амплитуда, период, сдвиг)]]).
- График $y = \sin(2x)$ — сжатие по горизонтали.

**Связь с физикой:**  
Форма волны в [[Гармонические колебания (физика)]] — это именно синусоида.
